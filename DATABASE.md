# База данных проекта "Plagiarism Detector"

## Тип СУБД
SQLite (файл db.sqlite3)

## Таблицы

### 1. core_corpusdocument (Корпус документов)
Назначение: хранение текстов для сравнения (книги, стихи, статьи).

| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER | Первичный ключ |
| title | VARCHAR(255) | Название документа |
| content | TEXT | Полный текст |
| file_name | VARCHAR(255) | Имя файла (если загружен) |
| created_at | DATETIME | Дата добавления |

### 2. core_checkhistory (История проверок)
Назначение: хранение всех результатов проверок текстов.

| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER | Первичный ключ |
| text_checked | TEXT | Проверенный текст (первые 500 символов) |
| uniqueness_score | FLOAT | Процент уникальности (0-100) |
| matched_count | INTEGER | Количество найденных совпадений |
| keywords | TEXT | Ключевые слова через запятую |
| stats | TEXT | Статистика в формате JSON |
| created_at | DATETIME | Дата проверки |

## Связи
Таблицы независимы (нет внешних ключей).
